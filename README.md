# CITC 停车行动终端 PWA

这是一个静态网页 App，不需要后端服务器。它负责计算免费停车挪车时间，iPhone 的快捷指令、日历或提醒事项负责可靠通知。

## 使用方式

1. 把 `parking-pwa` 文件夹部署到 GitHub Pages、Cloudflare Pages、Vercel 或 Netlify。
2. 用 iPhone Safari 打开部署后的 HTTPS 地址。
3. 点击 Safari 分享按钮，选择“添加到主屏幕”。
4. 到公司后打开 App，点击“我已入场，现在开始计时”。
5. 点击“加入日历”，把当天挪车提醒加入 iPhone 日历。
6. 每次挪车完成后，回到 App 点击“已完成挪车，重新计时”。
7. 下班离开后点击“结束今天”。

## iPhone 快捷指令建议

### 到达公司

- 触发条件：到达公司位置。
- 时间范围：07:00-10:30。
- 动作：打开 URL，填入 PWA 地址。
- 目的：提醒你到公司后立刻开始停车计时。

### 离开公司

- 触发条件：离开公司位置。
- 时间范围：17:30-21:00。
- 动作：打开 URL，填入 `PWA地址?action=end`。
- 目的：提醒你结束当天停车状态。

## 默认规则

- 免费停车：3 小时。
- 挪车耗时：15 分钟。
- 默认提前提醒：15 分钟。
- 午睡保护：13:00-14:00 尽量不安排挪车。
- 下班后：18:00 后不再安排挪车。

## 视觉方向

- 地点标识：CITC，来自 Cuiheng International Technology Center。
- 主色：红、橘、灰、黑。
- 风格：原创战术工业终端风，使用窄体数字、斜切模块、任务编号和警示色。
